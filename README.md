# 🩺 Insulin Levels and Diabetes Pedigree Function[DPF] Analysis

## 📌 Project Overview

This project investigates the relationship between insulin levels, Diabetes Pedigree Function (DPF), and diabetes outcomes. The analysis focuses on understanding how insulin measurements and genetic predisposition contribute to diabetes risk.

---

## 🎯 Objectives

- Analyze insulin level distributions.
- Study Diabetes Pedigree Function patterns.
- Compare diabetic and non-diabetic groups.
- Explore relationships among clinical variables.
- Generate meaningful healthcare insights.

---

## 🛠️ Tools & Technologies

- Power BI
- Python
- Pandas
- NumPy
- Microsoft Excel
- Statistical Analysis

---

## 📂 Dataset

**Source:** Pima Indians Diabetes Dataset

### Dataset Features

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age
- Outcome (Diabetic / Non-Diabetic)

  - Diabetes Outcome

### ⚕️ Outcome Analysis
- Diabetic vs Non-Diabetic Comparison
- Risk Factor Evaluation

---

## 📈 Key Insights

- Higher Diabetes Pedigree Function values are associated with increased diabetes prevalence.
- Diabetic individuals generally exhibit elevated glucose levels.
- Family history significantly contributes to diabetes risk.
- Multiple health indicators collectively influence diabetes outcomes.
